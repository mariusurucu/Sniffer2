Project Title:
UDP and TCP capture and count

Summary:
In this project, I was tasked to create a java application that would capture UDP or TCP packets and count how many such packets I had in a certain time frame or number of iterations. To carry out this project we used the external library pcap4j

Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.


Prerequisites and Installing:
To run my sample you need to install the following ->
-> Java version 8 : https://java.com/en/download/help/download_options.xml
->Install what java ide you like
->About Pcap4j: https://www.pcap4j.org/
